# 🌱 The Skill Garden — Interactive 3D Portfolio

A living, breathing 3D portfolio where your skills are represented as **growing plants**.
Add projects to earn XP, level up your skills, and watch your garden evolve from tiny
sprouts into legendary glowing trees.

## ✨ Features

- 🌿 **6 skill plants** arranged in a glowing circle — React, Three.js, Node.js, UI/UX, Python, CSS
- 🌱 **5 evolution levels** per plant — from sprout → seedling → bush → tree → legendary
- ⚡ **XP system** — add projects to earn XP and level up skills in real time
- 💥 **Particle bursts** on every level-up with scale animations
- 🌌 **Dynamic environment** — sky, fog, fireflies, and stars react to your total XP
- 🎥 **Smart camera** — orbit, zoom, auto-rotate when idle, click-to-focus on any plant
- 🏷️ **Floating skill labels** with live XP progress bars above each plant
- 💾 **Persistent garden name** saved to localStorage
- 📱 **Mobile friendly** — touch controls supported

## 🎮 How to Use

1. Open the site
2. Click **"Add Project"** on any project card in the bottom-left panel
3. Watch the linked skill plants grow and evolve
4. Keep adding projects to unlock higher plant levels
5. At 900+ XP the garden enters **Full Bloom** — fireflies swarm and stars twinkle

## 🌍 Environment Stages

| Total XP | Stage | Sky | Fireflies |
|----------|-------|-----|-----------|
| 0 – 200 | 🌑 Night | Deep black | None |
| 201 – 500 | 🌒 Dawn | Deep purple | 10 |
| 501 – 900 | 🌆 Dusk | Warm orange fog | 25 |
| 900+ | 🌸 Full Bloom | Deep teal | 50 |

## 🛠️ Tech Stack

| Tool | Usage |
|------|-------|
| [Three.js r128](https://threejs.org/) | 3D scene, geometry, animation loop |
| OrbitControls | Camera drag, zoom, auto-rotate |
| CSS2DRenderer | Floating HTML labels in 3D space |
| Vanilla JS | XP system, state, DOM overlays |
| Google Fonts (Inter) | Typography |

**Zero dependencies. Zero build tools. One HTML file.**
